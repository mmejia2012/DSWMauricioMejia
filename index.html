Estructura básica de html5 (etiquetas, atributos y valores)


BIENVENIDA

En este curso de html5 aprenderás la estructura básica de una página web, lo relacionado a las etiquetas esenciales para crear una web básica. En este curso, aprenderás a crear páginas web con las herramientas más fundamentales: HTML. Aprenderás a escribir tu propio código para la web, entendiendo cómo funciona por dentro y cómo diseñar una web, este curso puede suponer una buena base para comenzar en el mundo del Desarrollo Web Front-end. Este curso parte desde un conocimiento cero de HTML. Incluye las etiquetas fundamentales de HTML, sus atributos y valores o propiedades. Básicamente una página web esta compuesta de 2 partes, una cabeza <head> y un contenido o cuerpo <body>



Hernández, V.(May 4, 2018). e-Learning Master. recuperado de: http://elearningmasters.galileo.edu/2018/05/04/html5-en-cursos-virtuales/


OBJETIVO GENERAL

Desarrollar el OVA para el curso de Diseño de sitios web de la UNAD, que contenga lo relacionado al tema "Estructura básica de html5 (etiquetas, atributos y valores)" y que explique los conceptos más aplicados en el diseño y estructura básica de html5 para desarrollar una Web, permitiendo que el estudiante tenga las competencias necesarias para óptimo trabajo en Web. Primordialmente, consistirá en aplicar los conocimientos adquiridos sobre Web a un caso o cliente real, donde se tendrá en cuenta: Conceptuar sobre los conceptos de etiquetas, atributos y valores.

OBJETIVOS ESPECIFICOS

  • Conocer la estructura básica de una página web. (etiquetas, atributos y valores) Adquirir competencias y conocimientos de nuevos conceptos (cabeza, cuerpo)

  • Brindar elementos para la planeación, implementación y diseño de páginas web, en este OVA se establecen las bases y la metodología para apropiar el conocimiento necesario para desarrollar este tipo de trabajo.

  • Conocer a profundidad en las etiquetas básicas, su uso, sus variaciones y diferentes valores y como estas estructuran las páginas web que desarrollemos.
